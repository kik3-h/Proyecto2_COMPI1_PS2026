# Proyecto2_COMPI1_PS2026

## YFERA Framework_EH

Resumen del proyecto
YFERA Framework es un IDE web y transpilador educativo para la asignatura Organización de Lenguajes y Compiladores 1. El objetivo es implementar un flujo completo lexer → parser → AST → codegen que traduzca un lenguaje propio de componentes, estilos y lógica a HTML/CSS/JS, además de un sistema de archivos virtual y una consola de base de datos integrada para CRUD sobre SQLite/IndexedDB.
Proyecto finales del curso de Org. de lenguajes y compiladores 1 del Primer Semestre 2026

## Instalación rápida del proyecto

Requisitos previos

Node.js v24.x y npm compatible.

Angular CLI instalado globalmente.

Git y GitHub.
